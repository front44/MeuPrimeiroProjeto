# Meu Primeiro Projeto Git

Este repositório foi criado como parte de uma atividade acadêmica da faculdade.

## Objetivo
O intuito principal deste projeto é praticar os conceitos fundamentais de controle de versão, incluindo:
* Criação de repositórios.
* Gerenciamento de branches.
* Ciclo de vida de arquivos (add, commit e push).

## Tecnologias Utilizadas
* Git.
* GitHub.
